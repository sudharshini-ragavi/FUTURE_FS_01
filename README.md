# Personal Portfolio Website
This is my personal portfolio website developed as part of my Full Stack Web Development Internship.

## Features
- Interactive portfolio sections (Home, About, Skills, Projects)
- Contact form using PHP and MySQL
- Admin login system to view messages

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Backend Functionality
The contact form stores user messages in a MySQL database using PHP.
Admin can log in to view submitted messages.

## Note
The backend (PHP + MySQL) works in a local environment using XAMPP.
GitHub Pages supports only frontend features.
